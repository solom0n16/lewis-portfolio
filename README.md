# Lewis Buenaventura — Data Analytics Portfolio

This portfolio is a static HTML/CSS/JavaScript site hosted with GitHub Pages.

## Featured Project

### Glassdoor Data Analyst Job Market Analysis
A Python-based data cleaning and exploratory analysis project using 2,251 Glassdoor Data Analyst job postings.

Workflow:

1. Raw job-posting export
2. Data cleaning notebook
3. Cleaned CSV
4. EDA and insights notebook
5. Stakeholder-ready deck
6. Dashboard to follow
